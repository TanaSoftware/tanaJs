# tanaJs
Less code javascript library

tanaJs is small library for building easy web siteS for all kind of developers from beginners.

1. example

```javascript
var myWorld = 'world';
```
```html
<div style="display:none" clientSource="myWorld">
    <div>           
     <span>Hello </span> {{myWorld}}
    </div> 
 </div>
</html>
```
